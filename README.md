# tauri-shadcn-template

A tauri app template with shadcn-ui, and React.

## Installation

Install by cloning the project:

```bash
git clone https://github.com/languint/tauri-shadcn-template.git
```

## Development

To start the project, install dependencies:

```bash
pnpm i
```

And then run the development script

```bash
pnpm tauri dev
```
